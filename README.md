# 台湾历史统治时期地图

一个交互式的台湾历史地图可视化项目，展示台湾从古至今不同统治时期的行政区划变迁。

## 功能特点

- 🗺️ 五个历史时期的交互式地图
  - 荷兰西班牙统治时期 (1624-1642)
  - 明郑时期 (1662-1683)
  - 清治时期 (1683-1895)
  - 日据时期 (1895-1945)
  - 现代时期 (1945-至今)
- 📊 详细的历史信息展示
- 📚 史料摘录展示
- 🔍 地图缩放和交互功能
- 📱 响应式设计，支持移动端访问

## 技术栈

- HTML5 / CSS3
- JavaScript (ES6+)
- D3.js v7 - 数据可视化库
- TopoJSON - 地理数据格式

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器（推荐）：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx http-server
   ```
3. 访问 `http://localhost:8000`

## 历史时期说明

### 荷兰西班牙统治时期 (1624-1642)
- 荷兰以热兰遮城为中心，控制南部与西部
- 西班牙以圣萨尔瓦多城为中心，控制北海岸

### 明郑时期 (1662-1683)
- 郑成功收复台湾，建立明郑政权
- 设里疆界和屯垦地系统

### 清治时期 (1683-1895)
- 1683年清朝攻台，台湾纳入清朝版图
- 1885年升格为台湾省
- 三府一直隶州行政区划

### 日据时期 (1895-1945)
- 甲午战争后割让日本
- 五州三厅行政区划体系

### 现代时期 (1945-至今)
- 光复后的现代行政区划
- 22个县市行政体系

## 数据来源

- 历史地图数据来源于历史文献及学术研究
- 地理数据基于TopoJSON格式
- 行政区划信息参考《台灣歷史圖說 史前至一九四五年》
